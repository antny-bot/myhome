# 기능별 상세 문서 (Features Index)

> 구현된 기능별 상세 문서 모음. 작업 관련 문서 1개만 선택해 읽을 것. 전체 독서 금지.

## 사용법 (토큰 절약 규칙)

1. 표에서 작업 대상 기능 탐색.
2. 해당 행 "상세 문서" 1개만 열 것.
3. "관련 파일"은 문서 읽기 전 위치 확인용임.

## 기능 목록

| 기능 | 한 줄 설명 | 상세 문서 | 관련 파일(요약) |
|------|-----------|-----------|----------------|
| 지역 검색 / 주소 자동완성 | 지역명 입력 시 카카오 주소 검색으로 하위 후보 축소 선택. 키 없을 시 MCP 단일 검색 폴백 | [`region-search.md`](./region-search.md) | `server/addressSearch.ts`, `server/routes.ts`, `src/App.tsx` |

## 새 기능 문서를 추가할 때

- 표에 1행 추가(설명 + 링크 + 관련 파일).
- `docs/features/<기능명>.md` 작성.
- 상세 문서 자체 완결적 작성(해당 문서만으로 작업 가능하게).
- 필요시 `docs/OVERVIEW.md` "작업 유형별 필독 파일" 표에 행 추가.
