# 개발계획 체크리스트

> **규칙**: 모든 개발 작업은 `준비`, `진행중`, `완료` 체크리스트로 관리합니다.  
> 체크리스트 항목은 `## 모듈명` 섹션 아래에 추가하고, 각 항목은 다음과 같이 표기합니다.

## 📂 프로젝트 루트
- [ ] [준비] 프로젝트 초기화 (Git, CI, 배포 파이프라인)
- [ ] [준비] 기술 스택 결정 (Node, React, DB 등)
- [ ] [준비] 아키텍처 기본 설계

## 🖥️ 백엔드
- [ ] [준비] API 스펙 작성
- [ ] [진행중] 엔드포인트 구현
- [ ] [진행중] DB 스키마 설계 및 마이그레이션
- [ ] [완료] 단위/통합 테스트
- [ ] [준비] 보안 및 로그 설정

## 🎨 프론트엔드
- [ ] [준비] UI/UX 요구사항 정의
- [ ] [진행중] 와이어프레임 및 디자인 시스템 구축
- [ ] [진행중] 핵심 컴포넌트 개발
- [ ] [진행중] 상태 관리 구현
- [ ] [완료] 통합 테스트 및 레이아웃 검증
- [ ] [준비] 배포 파이프라인 설정

## 📅 진행 관리
- [ ] [준비] 스프린트 플래닝 템플릿 생성
- [ ] [진행중] 데일리 스탠드업 기록
- [ ] [진행중] 버그 트래킹 및 해결
- [ ] [완료] 회고 및 개선 계획
